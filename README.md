## 📥 Download

To download, do the following:

1. Go to the releases page (or click [THIS](https://github.com/oxyrv/FNaF2-PS3/releases))
2. Download the latest version (The .pkg file)
3. And then install the pkg on your console!

## ⚙ Building

To build, you need some requirements:
- A devkit console
- The official SDK
- GameMaker Studio 1 (other versions do NOT support PS3!)

(Note: The PlayStation 3 SDK is proprietary software owned by Sony.

You must legally own it to build this project.)
